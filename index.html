Features

Everything you need.
Nothing you don't.

BlockPanel packs a professional-grade toolset into a lightweight, zero-dependency desktop app.

One-Click Lifecycle

Start, stop, restart, and force-kill your server with single button clicks. Automatic restart on crash with configurable delay.

Live Console

Real-time log streaming direct from the Java process. Syntax-highlighted INFO, WARN, ERROR levels. Type commands inline.

JAR Manager

Browse and download every Minecraft version — releases and snapshots — directly from Mojang's CDN with live progress tracking.

Modrinth Mod Browser

Search, preview, and one-click install mods and plugins directly from Modrinth. Filter by version, loader, category, and sort by downloads.

World Map

Render an overhead view of your Minecraft world directly from region files. Pan, zoom, and double-click to copy teleport commands.

Intelligence & Heatmaps

Real-time telemetry tracking player retention, activity heatmaps, and death/teleport locations via SQLite. Optimize your realm with data-driven insights.

Local First · Private

Your data stays on your machine. No cloud tracking, no hidden accounts, no telemetry sent to us. Purely local server orchestration.

Studio-Grade Toolbar

A seamless, frameless experience. Native window management integrated directly into the thematic shell with a studio-engine status pill.

Studio Config Editor

Powered by the Monaco Engine. Syntax highlighting, error detection, and bracket matching for all your server.properties and JSON configs.

Network & DDoS Radar

Real-time bandwidth visualization. Track inbound/outbound packets and monitor connection health with high-fidelity system telemetry.

The Realm Edict

In-app historical record of all updates, features, and contributors. Stay informed on the evolving platform through a high-fidelity markdown viewer.

Quick Start

Up and running in minutes.

Three steps from zero to a live Minecraft server.

01

Download & Install

Grab the NSIS installer or MSI package below. Run it — no dependencies, no runtimes, no bloat. BlockPanel is fully self-contained.

02

Configure Your Server

Open Settings. Set your server directory and download a Minecraft JAR from the built-in manager. BlockPanel handles EULA and server.properties automatically.

03

Optimize with Intelligence

Click the Start button. beyond basic logs, use the new Intelligence tab and Network Radar to track player loyalty and system health in real-time.

Portable Mode

Take your server anywhere.

BlockPanel is designed to be truly portable. Drop the app and a portable.txt file in the same folder — all configs, JARs, and backups stay right there.

  • Run from a USB drive
  • Zero AppData pollution
  • Project-scoped installations
  • No admin rights needed
